ASSIGNMENT

The assignment is: Design and create a website to read stories online. The website becomes responsive. Visitors must be able to find and read stories. Visitors must also be able to leave something themselves, such as a reaction, a like, or a photo.

In this project I learned how to design and create a responsive website. I have learned how I can substantiate all design choices with design research for the design and interaction, and how I can make iteratively a design using HTML. CSS and JavaScript. Project WEB is an individual project where I have created and created the design, technology and interaction.
